Tree Traverse
-----------------

-DFS Traversal
(1) PreOrder  (DLR)	BST:Not Sorted	 Root Node:Location: First Node
(2) InOrder   (LDR)	BST:Sorted	     Root Node Location: Middle Node
(3) PostOrder (LRD)	BST:Not Sorted	 Root Node Location: Laste Node

  ** Above 3 tree traversal are depth first search traversal 
  
(4) Level order  is Breadth first search traversal  -BFS
